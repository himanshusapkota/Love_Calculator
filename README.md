# ❤️ Love Calculator

A fun and interactive Love Calculator built with **HTML, CSS, and JavaScript**. Enter two names and discover their love compatibility percentage with beautiful animations, romantic visuals, and personalized messages.

## ✨ Features

- ❤️ Love compatibility percentage generator
- 💖 Romantic glassmorphism UI
- 🎈 Floating heart animations
- 📊 Animated progress bar
- ⏳ Loading effect before results
- 💬 Dynamic compatibility messages
- 🔄 Reset button
- 📱 Fully responsive design
- 🚀 Beginner-friendly code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

```text
love-calculator/
│
├── index.html
├── style.css
└── script.js
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your browser.
4. Enter two names and click **Calculate**.

## 📸 Preview

The application includes:

- Animated floating hearts
- Modern pink gradient background
- Glassmorphism effect card
- Love percentage meter
- Personalized love messages

## ⚠️ Disclaimer

This project is created for entertainment purposes only. The results are randomly generated and do not represent actual relationship compatibility.
